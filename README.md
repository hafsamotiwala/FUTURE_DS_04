# E-Commerce Funnel & Conversion Analysis

## Project Overview
This project analyzes a 6-month Direct-to-Consumer (D2C) e-commerce dataset to evaluate customer behavior across the complete marketing funnel — from website visit to purchase completion. The objective is to identify drop-off points, measure conversion performance, and provide actionable business recommendations to improve revenue.

---

## Project Components

### 1. Codebook
Contains dataset structure, column descriptions, and data definitions used throughout the analysis.

### 2. Business Report
A concise, non-technical summary explaining key findings, behavioral patterns, performance insights, and strategic recommendations.

### 3. Excel Dashboard
An interactive dashboard built using Pivot Tables and slicers, showcasing:
- Funnel stage performance  
- Conversion rates by channel, device, and campaign  
- Monthly trends  
- Key performance indicators (KPIs)

---

## Key Objectives
- Identify major funnel drop-off stages  
- Compare marketing channel effectiveness  
- Analyze mobile vs desktop behavior  
- Evaluate campaign performance  
- Provide practical strategies to increase conversion and revenue  

---

## Tools Used
- **Python (Pandas)** – Data cleaning and funnel analysis  
- **Excel** – Dashboard creation and visualization  

---

## Outcome
This project demonstrates applied marketing analytics, funnel optimization analysis, and business-focused insight generation aimed at improving conversion rates and marketing ROI.
